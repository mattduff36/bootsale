# Design Assets

This directory contains the original design files and source assets for carbootsale.com.

## Contents

- **CarBoot Sales Stuff/** - Original logo concepts and variations
- **Carboot_Icons/** - Full icon set (AI, EPS, PNG formats)
- **Carbootsale/** - Logo files in various formats
- **New-Square-Logo/** - App logo with .com branding
- **New-Square-Logo-without_dot_com/** - App logo variations without domain

## For Developers

These files are **NOT** served to users. They are kept for reference and future design work.

**Runtime assets** are copied to:
- `public/brand/` - Logos used on the website
- `public/icons/` - Icons used in the UI

## Source Files

Many files are in Adobe Illustrator (.ai) and Photoshop (.psd) formats.
To edit these, you'll need:
- Adobe Illustrator
- Adobe Photoshop

Or use open-source alternatives:
- Inkscape (for .ai/.eps files)
- GIMP (for .psd files)

## Color Palette

From the icon set:
- **Brown**: `#4A3428` (primary brand color)
- **Bright Green**: `#7FFF00` (accent color)
- **Light Blue**: `#87CEEB` (background in logos)

## Icon Categories

The icon set includes:
- Search, map, directions
- Indoor/outdoor venue markers
- Amenities: toilets, disabled toilets, refreshments
- Contact: phone, website
- UI elements: yes/no indicators, home
